## Tech Debt (non-blocking)

### HERMES-21: postinstall script wastefulness
- **Location:** sibling repo `package.json` `scripts.postinstall`
- **Issue:** Runs `npm --prefix servers install` on every consumer install,
  but the published bundle is fully bundled (esbuild `--packages=bundle`)
  with zero runtime deps. Installs dev/test deps the consumer never uses.
- **Severity:** Wasteful, not broken
- **Suggested fix:** Remove `postinstall` script in a v1.2.1 patch (or fold
  into the next coordinated release)

### HERMES-17: Env-leak workaround
- **Location:** test execution environment
- **Issue:** `CHATLYTICS_API_KEY` / `CHATLYTICS_API_URL` / `CHATLYTICS_SESSION`
  can leak from the orchestrator shell into pytest, causing the 10
  auth-related tests to fail because env vars override the fixture-injected
  values.
- **Severity:** Environmental quirk, not a production issue
- **Suggested fix:** Add `monkeypatch.delenv(...)` for these three keys at
  session-start in `tests/conftest.py`. Trivial; pick up in v3.1.
